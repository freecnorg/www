Notepad++ 文本编辑工具

2021-02-01 #编辑器 #Windows

Notepad++是一个免费的源代码编辑器和记事本替换，支持多种语言。在 MS Windows 环境中运行，其使用受GNU 通用公共许可证管辖。

基于强大的编辑组件Scintilla，记事本+写在C++并使用纯 Win32 API 和STL，这确保了更高的执行速度和更小的程序大小。通过优化尽可能多的例程，同时不失去用户友好性，Notepad++ 正在努力减少全球二氧化碳的排放。当使用较少的 CPU 功率时，PC 可以降低功耗并降低功耗，从而形成更环保的环境。
